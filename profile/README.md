# ReadCube Paper Manager

**ReadCube Paper Manager** is a professional reference management and PDF organization platform designed for researchers, students, and academics. As a powerful **ReadCube Research Library**, it provides access to over 30 million free full-text articles from PubMed, Google Scholar, and institutional subscriptions. This **ReadCube PDF Annotator** lets you highlight text, add sticky notes, draw shapes, and create color-coded tags directly on documents. With **ReadCube Reference Tool**, you can generate citations in APA, MLA, Chicago, Harvard, and thousands of journal-specific formats with one click.

## Features

- **Smart Organization**: The **ReadCube Paper Manager** auto-extracts metadata from imported PDFs.
- **Full-Text Access**: Use **ReadCube Research Library** to find and download free versions of paywalled papers.
- **Annotation Suite**: As a **ReadCube PDF Annotator**, it includes sticky notes, highlighting, and freehand drawing.
- **Citation Engine**: This **ReadCube Reference Tool** integrates with Microsoft Word and Google Docs.
